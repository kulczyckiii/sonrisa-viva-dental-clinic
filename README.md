# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).

## Expanding the ESLint configuration

If you are developing a production application, we recommend using TypeScript with type-aware lint rules enabled. Check out the [TS template](https://github.com/vitejs/vite/tree/main/packages/create-vite/template-react-ts) for information on how to integrate TypeScript and [`typescript-eslint`](https://typescript-eslint.io) in your project.

# Sonrisa Viva Dental Clinic — Web Application

📝 Descripción del Proyecto

Sonrisa Viva es una solución web integral diseñada para digitalizar la experiencia del paciente en clínicas dentales. Esta aplicación resuelve la ineficiencia del agendamiento manual mediante un sistema de reserva de citas intuitivo, una presentación profesional de servicios médicos y una arquitectura robusta orientada al rendimiento.

🚀 Funcionalidades ClaveLanding Page de Alta Conversión: Presentación clara de la propuesta de valor, servicios de salud y equipo médico.Módulo de Agendamiento Inteligente: Formulario con validación en tiempo real y selección de horarios simulados para optimizar la captación de pacientes.Persistencia Local: Uso de localStorage para prevenir la pérdida de datos del usuario durante el proceso de registro.Diseño Full Responsive: Experiencia optimizada para dispositivos móviles, tablets y desktop.

🛠️ Stack TecnológicoFrontend: ReactJS (Vite).Estilos: Tailwind CSS (Arquitectura basada en componentes y diseño semántico).Enrutamiento: React Router DOM para navegación fluida sin recarga de página.Arquitectura: Estructura organizada por capas: Presentation, Domain Logic, State Management e Infrastructure.
